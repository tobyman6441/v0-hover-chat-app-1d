FEATURE 2: FLOATING FEEDBACK BUTTON
--------------------------------------------------------------------------------

DESCRIPTION:
Add a floating feedback button in the bottom-right corner of the app that
allows users to send feedback via email to support@hover.ninja.

IMPLEMENTATION DETAILS:

1. BUTTON APPEARANCE:
   - Floating button fixed to bottom-right corner
   - Contains an icon (message/feedback icon) and text "Feedback"
   - Styled with the app's design system (rounded, shadow, etc.)
   - Should be visible on all pages but not obstruct content

2. BUTTON BEHAVIOR:
   - On click: Opens a mailto link to support@hover.ninja
   - mailto:support@hover.ninja?subject=Hover%20Ninja%20Feedback

3. COMPONENT STRUCTURE:
   - Create a FeedbackButton component
   - Place it in the root layout so it appears on all pages
   - Position: fixed, bottom-right with some padding (e.g., bottom-4 right-4)

4. STYLING SUGGESTIONS:
   - Background: Primary color or neutral
   - Border radius: rounded-full or rounded-lg
   - Shadow: shadow-lg for floating effect
   - Padding: comfortable touch target (at least 44x44px)
   - Z-index: High enough to float above content (z-50)

5. ICON:
   - Use a feedback/message bubble icon from Lucide React
   - Suggested icons: MessageCircle, MessageSquare, HelpCircle

EXAMPLE CODE STRUCTURE:
```tsx
// components/feedback-button.tsx
"use client"

import { MessageCircle } from "lucide-react"
import { Button } from "@/components/ui/button"

export function FeedbackButton() {
  return (
    <a
      href="mailto:support@hover.ninja?subject=Hover%20Ninja%20Feedback"
      className="fixed bottom-4 right-4 z-50"
    >
      <Button className="rounded-full shadow-lg gap-2">
        <MessageCircle className="h-4 w-4" />
        Feedback
      </Button>
    </a>
  )
}
```

PLACEMENT:
- Import and render in app/layout.tsx (root layout)
- Place after the main content so it floats above everything
